# `boards/stm32-nucleo-f401re/` — STMicroelectronics Nucleo-F401RE

- **MCU**: STM32F401RE
- **Architecture**: ARM Cortex-M4 84 MHz
- **Flash**: 512 KB
- **RAM**: 96 KB
- **Toolchain**: stm32-hal
- **Flash protocol**: st-flash (ST-Link)
