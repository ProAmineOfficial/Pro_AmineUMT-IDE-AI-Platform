# Documentation

The Pro_AmineUMT IDE with AI Platform documentation is structured for four
audiences: new users, application developers, platform contributors, and
standards implementers.

## Structure

- `getting-started/` — new user onboarding.
- `architecture/` — system architecture, AIA Engine internals, source map.
- `api-reference/` — the UMT VR API surface (all 45 interfaces).
- `guides/` — task-oriented how-to guides.
- `tutorials/` — step-by-step learning material.
- `style-guides/` — language style guides for contributors.
- `faq.md` — frequently asked questions.

## Building the documentation

Documentation is authored in Markdown. The static site is generated by
MkDocs Material and published to `https://docs.pro-amine.com/umt/` on every
push to `main`.
