# Packages (`packages/`)

Reusable packages shipped independently of the applications:

- `framework/` — the UMT Framework (aliased in the SDK as
  `Pro_AmineUMT Framework` = UMT SDK VR API layer).
- `toolchains/` — bundled toolchain manifests and download hashes.
- `libraries/` — first-party libraries for common tasks (JSON parser,
  MQTT client, filesystem, cryptography).

## Publication

Packages are published to the internal `@pro-amineumt` npm scope and the
UMT Package Registry at `packages.pro-amine.com`.
