# Partners (`partners/`)

Industry partners of the Pro_AmineUMT IDE with AI Platform.

## Categories

- `chipmakers/` — organizations adopting the UMT BGA 16×16 Package
  Substrate Standard at the package substrate stage.
- `distributors/` — organizations selling or distributing NanoKit boards
  or UMT-branded merchandise under license.
- `standards-bodies/` — engagement with JEDEC, IPC, and equivalent
  standards organizations.
