# Examples (`examples/`)

Complete example projects, each self-contained.

- `adc_read/` — the canonical VR_ADC4 example (potentiometer).
- `led_blink/` — the classic first program.
- `uart_echo/` — bidirectional serial.
- `pwm_led/` — smooth LED brightness.
- `i2c_scan/` — enumerate devices on an I²C bus.
- `dual_role/` — the same pin as ADC (activate) and as LED (deactivate).
- `multi_target/` — a single `.umt` proven to run on ESP32, UNO, and Pico.

Every example builds unmodified on every supported target unless the
example's `umt.json` restricts it.
