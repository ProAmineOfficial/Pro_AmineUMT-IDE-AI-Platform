# `apps/dashboard/`

Web dashboard for connecting projects to AdvancedHMI (C#), Flutter apps, and MQTT brokers. Runs on the developer's machine or on a remote host.

## Build

```sh
cd apps/dashboard
npm install
npm run build
```

## Depends on

- `core/aia-engine`
- `core/kernel`
- `core/source-map`
