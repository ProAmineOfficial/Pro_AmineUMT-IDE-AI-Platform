# `apps/cli/`

Command-line interface — `umt build`, `umt flash`, `umt debug`, `umt monitor`, `umt clean`, `umt new`.

## Build

```sh
cd apps/cli
npm install
npm run build
```

## Depends on

- `core/aia-engine`
- `core/kernel`
- `core/source-map`
