# `apps/simulator/`

Visual block-based simulator, similar to Node-RED. Renders the target board and attached components. Bidirectional: code updates diagram, diagram updates code.

## Build

```sh
cd apps/simulator
npm install
npm run build
```

## Depends on

- `core/aia-engine`
- `core/kernel`
- `core/source-map`
