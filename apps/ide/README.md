# `apps/ide/`

Visual Studio Code extension providing the editor, debugger, monitor, project manager, AI Bashir sidecar, and Genius Guides panel.

## Build

```sh
cd apps/ide
npm install
npm run build
```

## Depends on

- `core/aia-engine`
- `core/kernel`
- `core/source-map`
