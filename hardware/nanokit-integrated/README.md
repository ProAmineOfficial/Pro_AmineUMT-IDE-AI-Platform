# `hardware/nanokit-integrated/`

The NanoKit Integrated Model ESP32 V2020 SMD/THT — 40-pin development board. Designed in Algeria, manufactured in China. Cloner mechanism maps ESP32 pins to UMT BGA 16×16 VR addresses.

## Structure

- `schematic/` — KiCad schematic files.
- `layout/` — KiCad PCB layout.
- `spec/` — datasheet and mechanical drawings.
- `photos/` — reference photographs (placeholders here; released
  versions carry actual imagery).
