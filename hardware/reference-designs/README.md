# `hardware/reference-designs/`

Application-level reference designs demonstrating the UMT Platform on common embedded use cases: motor controllers, sensor gateways, edge AI accelerators.

## Structure

- `schematic/` — KiCad schematic files.
- `layout/` — KiCad PCB layout.
- `spec/` — datasheet and mechanical drawings.
- `photos/` — reference photographs (placeholders here; released
  versions carry actual imagery).
