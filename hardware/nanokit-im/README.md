# `hardware/nanokit-im/`

The NanoKit-iM — **(MCU or SoC) Integrated Modules** — swappable modules on a BGA 16×16 substrate. Same PCB, different silicon die.

## Structure

- `schematic/` — KiCad schematic files.
- `layout/` — KiCad PCB layout.
- `spec/` — datasheet and mechanical drawings.
- `photos/` — reference photographs (placeholders here; released
  versions carry actual imagery).
