# Project Templates (`templates/`)

Starter templates created by `umt new`.

- `starter/` — the minimum viable project (`src/MyProject.umt`, `umt.json`).
- `multi-file/` — multi-module project with `src/`, `include/`, `lib/`.
- `with-tests/` — project including `test/` and CI pre-configured.

## Usage

```sh
umt new MyProject --template starter --target nanokit-esp32
```
