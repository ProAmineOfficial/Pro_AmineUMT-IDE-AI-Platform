# Prototype (`prototype/`)

Research and experimental work that is not yet part of the stable
platform. Everything here is subject to breaking changes without notice.

## Subdirectories

- `research/` — early-stage investigations, papers, and design sketches.
- `experimental/` — code that runs but is not production-ready.

## Graduation

An artifact graduates from `prototype/` by becoming an ADR proposal.
Once the ADR is accepted, the artifact moves to its permanent home
under `core/`, `apps/`, `hardware/`, or `standards/`.
